# 🤖 Rule-Based Smart Chatbot

A simple **Rule-Based Smart Chatbot** developed using Python. This chatbot interacts with users through predefined rules and responds to various queries such as greetings, programming languages, Artificial Intelligence, Machine Learning, books, career guidance, date and time, jokes, and basic calculations.

## 📌 Features

* Interactive command-line chatbot
* Rule-based conversation using `if-elif-else`
* Greets users with random responses
* Displays current date and time
* Answers questions about Python, AI, and Machine Learning
* Recommends books and courses
* Suggests project ideas
* Provides career guidance
* Performs basic mathematical calculations
* Tells programming jokes
* Displays a help menu with available commands
* User-friendly interface

## 🛠️ Technologies Used

* Python 3
* Built-in Python Modules:

  * `datetime`
  * `random`

